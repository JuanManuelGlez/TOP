Preguntas & respuestas

¿Cuál es la diferencia entre Internet y la World Wide Web?

El internet es la red de computadoras conectadas entre si, mientras que la World Wide Web es un conjunto de paginas al que puedes acceder mediante internet.

¿Cuáles son las partes de un URL?

Las partes de la url son las siguientes:

  • Protocolo HTTPS
  • Subdominio
  • Dominio
  • TLD
  • Ruta
  • Parámetro
  • Etiqueta
¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
  • GET : solicita un recurso del servidor web
  • HEAD : solicita solo el encabezado, sin el archivo
  • POST : sirve para enviar al servidor web paquetes grandes de datos, adjuntandose en el encabezado html
  • PUT : modifica todo el recurso completo
  • PATCH : modifica parte de un recurso
  • DELETE : eliminar un recurso
¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

El protocolo HTTP proporciona varias formas de realizar una solicitud; Los datos del formulario HTML se pueden transmitir a través de un número de diferentes queridos, los más comunes de los cuales son el método GET y el método POST

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

Código 200: esto no es un error HTTP, sino todo lo contrario. Significa que la página ha cargado de forma correcta

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Un Error 404 es el código HTTP que envía el servidor al usuario cuando la URL a la que está intentando acceder no existe

Usualmente, esto indica que el servidor no puede encontrar un mejor código de respuesta del tipo 5xx, por lo que si es responsabilidad del desarrollador.

¿Qué componentes de estructura y estilo tiene una tabla?

Estructura de una tabla (table) Una tabla HTML (table) es un conjunto de celdas (td o th) organizadas en filas tr) que a su vez se organizan en grupos de filas (thead, tbod> o foot). Además, la tabla puede tener una leyenda (caption) y hacer referencia a las columnas (col y colspan)..
